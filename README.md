# MySnakeGame
MySnakeGame

Sử dụng thư viện SDL
Trò chơi:
-Sử dụng các phím mũi tên để di chuyển
-Ăn 1 lần rắn dài thêm 1 ô và được thêm 10 điểm
-Khi đầu rắn đâm vào thân thì hiện GAME OVER.
